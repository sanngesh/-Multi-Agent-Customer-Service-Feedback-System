# Executive Summary

## Multi-Agent Customer Service Feedback System

This project delivers an AI-powered multi-agent system that automates the review of customer service chat transcripts in a banking context. Given three live chat transcripts between a support representative and customers, the system identifies problematic statements, explains why each is problematic, and proposes professional alternatives — all compiled into a formatted Word document titled "Case Feedback."

The solution employs a four-agent pipeline built in Python and powered by OpenAI's GPT-4o model. Agent 1 (Case Analyzer) performs deep analysis of each transcript against seven quality categories — tone, clarity, empathy, grammar, professionalism, problem resolution, and accuracy — quoting problematic statements verbatim. Agent 2 (Feedback Generator) produces structured explanations using category language and substantively different professional alternatives. Agent 3 (Quality Verifier) cross-checks all output against the original transcripts for accuracy, completeness, and quality, triggering automatic re-generation when issues are detected. Agent 4 (Document Compiler) assembles the verified feedback into a Word document with bold section headings, 1.5 line spacing, and professional formatting.

Key engineering decisions include structured JSON output mode for reliable parsing, temperature tuning per agent role (0.1–0.3), explicit guardrails against PII leakage and overpromising, and a self-correcting verification loop with up to two retry attempts per case. The pipeline successfully processed all three transcripts, identifying 15 feedback items in total and demonstrably improving output quality through its iterative verification cycle.

The system is evaluated against a 100-point rubric spanning correctness, completeness, clarity, format, and usefulness. Risk mitigation addresses hallucination, contextual misinterpretation, compliance concerns, and data privacy through automated verification, human review checklists, and prompt-level constraints. While the AI accelerates and standardises the feedback process, human oversight remains essential for institutional context, fairness, and final approval before any feedback reaches its intended audience.
